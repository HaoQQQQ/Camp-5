# 專題內容：森活營家-露營入口網站

一個方便尋找與預訂營地的平台，另可以購買商品、集點、參加活動等等，同時也提供營地主自行上架營地。

個人主要負責部分：營地細節頁、結帳流程，前端部分使用React和Bootstraps，後端則使用Node.js與資料庫做連結。
